# 🎓 Udemy Clone - Frontend

A responsive and modern Udemy-inspired online learning platform built using HTML, CSS, and JavaScript. This project replicates the user interface of Udemy, providing an engaging learning experience with course browsing, categories, and responsive design.

## 🚀 Features

- Responsive design for mobile, tablet, and desktop
- Modern and user-friendly UI
- Course listing section
- Categories and navigation menu
- Featured courses section
- Search functionality UI
- Student reviews section
- Footer with useful links
- Smooth animations and hover effects

## 🛠️ Technologies Used

- HTML5
- CSS3
- Responsive Web Design
